## Features
- Plays music
  - Has basic control features, like play/pause, skip forward/skip back, loop library, repeat song
- Adds music
  - If yt-dlp is installed, will download songs via URL
- Lists music
- Deletes music
- Saves everything to `~/.music_library` for ease of use
